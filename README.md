# Phenakistiscope Generator / Responsive Version

音楽機能なし版をもとに、スリット付き円盤のプレビューと保存機能を追加し、PC・スマホ両対応のレスポンシブ表示にした版です。

## p5.js Web Editor で使う場合

`sketch.js` の中身を p5.js Web Editor の `sketch.js` にそのまま貼ってください。

## GitHub Pages で使う場合

以下をリポジトリ直下に置いてください。

- index.html
- style.css
- sketch.js

すでに GitHub Pages を有効化している場合は、この3ファイルを上書きコミットすれば反映されます。
